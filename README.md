# **mhgame**
## Juego 2d de carreras entre obstaculos

https://mrivero40.github.io/mhgame/

Juego 2d de carreras entre obstáculos, desarrollado en javaScript con canvas. El fantasma (player) tiene que llegar al mundo sorteando los obtáculos (bombas). El jugador puede usar las flechas del teclado, o optar por los botones en pantalla.

Especificaciones:
- canvas tamaño dinámico.
- localStorage para guardar record y borrarlo.
- lógica para calcular la posición de los elementos a partir de las dimensiones del navegador.
- mapas renderizados a partir de array bidimensionales según convención establecida en maps.js

... seguiremos agregando algunas mejoras y arreglos...

Tecnologías aplicadas: HTML - CSS - JAVASCRIPT
